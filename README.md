# Meu Catálogo de Filmes

Este é um projeto React simples para criar e gerenciar uma lista de filmes.  
Você pode adicionar novos filmes à lista usando um campo de entrada interativo.

---

## Tecnologias usadas

- React
- JavaScript
- HTML / CSS
